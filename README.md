# DoIt Cadastro de Atividades
Projeto de cadastro de Atividades desenvolvido com as tecnologias Java e MySQL para fins de estudo. <br/>

## Tecnologias e ferramentas Utilizadas

- Java - Desenvolvimento do backend;<br/>   
  - Swing - Criação das Interfaces;<br/>

- MySQL - Banco de Dados;
  
- Git/Github - Versionamento do projeto;
- Figma - Criação dos Wireframes; <br/>

### Arquitetura do projeto
 - Por meio da utilização de DAO (Data Access Object) se separa a interface de métodos, facilitando a manutenção e organização;

 - Interfaces gráficas criadas por meio de elementos Swing.

 - CRUD para o cadastro e verificação de usuários, além do cadastro, exibição, e exclusão de atividades.

### Funcionamento do programa
A principal função é o cadastro de usuários e atividades, exibindo aquelas que foram organizadas por cada um deles, como uma lista de tarefas. Posteriormente, é possível a exclusão de cada uma dessas mesmas atividades por meio da ação do usuário.
